推荐展示的三个项目

数据爬取与线索生成

BBQ / Catering Lead Generation System

一个面向商家线索生成的自动化管道，组合 Serper 搜索、Google Maps/Facebook 页面解析、数据去重与 Supabase 存储。

  • 周定时任务抓取、关键字生成、结果解析、商机线索存储
  • FastAPI 提供 GET /leads、GET /stats 等浏览接口
  • 支持 Supabase 持久化与后端去重逻辑

YouTube 评论分析

YouTube Pain Point Pipeline

基于关键字抓取候选视频、评论采集与 LLM 过滤，最终生成痛点分析与商业输出建议。

  • 自动筛选候选视频、Top/Latest/痛点评论采样
  • 评论分类、信息价值过滤、批次分析与最终报告
  • 可导出 CSV，并支持 Supabase 持久化
项目详情
Shopify 自动化上传

Shopify 批量商品上传系统

竞品分析 + LLM 优化 + Shopify CSV 一键上传的全自动商品管道，支持两条路径：从竞品 URL 爬取或从 SKU 表格生成。

  • Playwright 爬取 Amazon 竞品页面，提取标题/价格/图片/规格
  • Ollama LLM 生成 SEO 标题 + HTML 描述，输出标准 Shopify CSV
  • Serper API 搜索增强 + 三级分类，支持批量处理数百商品
